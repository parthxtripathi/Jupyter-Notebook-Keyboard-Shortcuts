# Jupyter-Notebook-Keyboard-Shortcuts

## Command Mode (press Esc to enable)

| Enter | enter edit mode |
| ----- | --------------- |
| Shift-Enter | run cell, select below |
| Ctrl-Enter | run cell |
| Alt-Enter | run cell, insert below |
| Y | to code |
| M | to markdown |
| R | to raw |
| 1 | to heading 1 |
